Detailed Description

template<typename T>
class soundmath::BlockVocoder< T >

Advanced phase-vocoder with formants preservation and phase-sync processing.

The input of the process function is a workspace ready for FFT; the input is wksp while the output is wksp2; note the the input is also transformed during processing. The data in wksp MUST be windowed correctly and the output wksp2 must be overlapp-added.
